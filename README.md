## 📊 Overview
This project presents a comprehensive analysis of a bank's customer data, aiming to uncover key insights into customer behavior and identify factors influencing the success of marketing campaigns for term deposits.

By examining customer demographics, financial habits, and interactions with previous marketing efforts, we reveal the profiles of successful subscribers and provide actionable recommendations for optimizing future strategies.

## 🚀 Key Features
- **Customer Profiling**: Detailed analysis of age, job, marital status, and education.
- **Financial Impact**: Assessment of how housing and personal loans affect subscription rates.
- **Temporal Analysis**: Identification of the best months and days to contact potential clients.
- **Channel Effectiveness**: Comparison between cellular and telephone communication.
- **Interactive Visualizations**: Dynamic charts powered by Plotly for deep-dive analysis.

- **Programming Language**: Python

## 📦 Installation & Setup

1. **Clone the repository**:
   ```bash
   git clone https://github.com/helqadiri03/bank-data-analysis.git
   cd bank-data-analysis
   ```

2. **Create a virtual environment** (optional but recommended):
   ```bash
   python -m venv venv
   source venv/bin/activate  # On Windows: venv\Scripts\activate
   ```

3. **Install dependencies**:
   ```bash
   pip install -r requirements.txt
   ```

4. **Run the application**:
   ```bash
   streamlit run Main.py
   ```
